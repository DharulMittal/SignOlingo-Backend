- Default: 20 items per page
- Max: 100 items per page
- Cursor-based pagination for large datasets (future)

### Caching Strategies (Future)
- Cache dashboard data (5 min TTL)
- Cache achievement definitions
- Cache alphabet/word content
- Invalidate on progress updates

- Horizontal scaling with PM2
- Session affinity not required (JWT stateless)
- Database read replicas
